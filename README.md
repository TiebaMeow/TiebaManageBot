# TiebaManageBot

_基于 [Nonebot2](https://github.com/nonebot/nonebot2) 的贴吧管理机器人_

</div>

## 简介

欢迎使用贴吧管理 bot，本 bot 旨在为吧主和各位吧务在吧务群内提供一种指令清晰、交互方便的贴吧管理方式，功能包括但不限于封禁拉黑等基础吧务操作、循封管理、申诉推送与处理、信息查询与录入等。

## 安装依赖

```shell
uv sync
```

## 运行

```shell
uv run nb run
```
